# 🧮 Data Science & AI Math Vault

Welcome to my personal Mathematics repository! 🚀 

As a BSc student passionate about tech, I realized that understanding the mathematical foundation is the key to mastering Data Science, Machine Learning, and AI. Instead of keeping physical notes, I created this digital vault. 

This repository serves as a structured, easily searchable, and ever-growing collection of mathematical formulas, concepts, and theorems. It bridges academic learning with real-world technical applications.

### 🎯 Core Objectives:
* **Digital Cheat Sheets:** Quick references for complex formulas.
* **AI/ML Focus:** Special emphasis on Linear Algebra, Calculus, and Probability Theory used in algorithms.
* **Continuous Learning:** Documenting new concepts semester by semester.
